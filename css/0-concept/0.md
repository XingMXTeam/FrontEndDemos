## 基本术语

* 属性
* 属性值 1px solid rgb(0,0,0) 冒号后面的所有
* 值  1
* 关键字 transparent solid inherif 
* 变量  currentColor
* 长度单位 绝对： px 相对字体： em ex / rem ch  相对视区： vh vw vmin vmax
* 功能符 rgba hsla url calc 
* 声明 属性值+属性名
* 声明块 { 声明 ... }
* 规则集 .head { ... }
* 选择器 .class #a [title]{}
* 关系选择符 > ~（后续兄弟） +（相邻兄弟） 空格
* 规则 @media @font-face
